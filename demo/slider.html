API


            
The following are the essential invocations for working with Lectern.Slider:

            

                
$(containers).lectern('slider', {options})

                
$(containers).lectern('slider', 'create', {options})

                

                    
Create a new slider.

                


                
$(containers).lectern('slider', 'active')

                

                    
Convenience method to retrieve a jQuery collection of active slides.

                


                
$(containers).lectern('slider', 'next')

                
Programmatically advance a slider to the next slide.


                
$(containers).lectern('slider', 'prev')

                
Programmatically rewind a slider to the previous slide.
